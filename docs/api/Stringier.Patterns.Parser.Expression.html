Remarks

This provides both facilities to parse a source text into an Expression through Parse(ref Source) and TryParse(ref Source, out Expression), along with their respective overloads, as well as an interpreter of that parsed expression, through Evaluate(). Essentially, through this type, there is a fully implemented DSL.

This type is intended to be used to provide a generic and universal DSL for Stringier's Pattern type. The Expression syntax, parser, and interpreter are fully enclosed within Stringier.Patterns.Parser and fully exposed through this type. By providing a default library for this, both first-party and third-party tooling can utilize the same system, and therefore behave identically.
