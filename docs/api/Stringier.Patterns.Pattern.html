Mutable()

Get an empty mutable pattern.

Declaration
public static Pattern Mutable()
Returns
Type Description
Pattern

An empty mutable pattern.
Remarks

This provides a pattern which mutates in-place, which is dangerous. However, it is required when using the target/jumper system as left-recursion and mutual-recursion are not possible without it. Seal() should be called when done, to prevent further modification than intended, which can have disasterous consequences.
